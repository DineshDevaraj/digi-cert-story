{{ page }}. Rothchild bank's priv key, pub key & csr

  1. Generate private key. openssl genrsa -out rothchild.priv 2048
  2. Extract public key from private key. openssl rsa -in rothchild.priv -pubout -out rothchild.pub
  3. Generate (CSR) Certificate Sign Request. openssl req -new -key rothchild.priv -out rothchild.csr -subj "/C=US/ST=California/L=San Francisco/O=Rothchild Bank Inc./CN=rothchildbank.com"
  4. View the csr file generate in previous step. openssl req -in rothchild.csr -text -noout -verify
  5. As a next step go ahead and submit rothchild.csr to MicroCert.
