{{ page }}. Add certificate to bank's web-server

  1. You need to add rothchild.crt file to web-server.
  2. The steps will be different for different web-servsers. Here I have given the steps for Nginx.
  3. Copy rothchild.crt and rothchild.priv to standard folder. cp ./rothchild.crt /etc/nginx/ssl/rothchildbank.com
    cp ./rothchild.priv /etc/nginx/ssl/rothchildbank.com
  4. Add the following lines to Nginx config file. ssl_certificate /etc/nginx/ssl/rothchildbank.com/rothchild.crt
    ssl_certificate_key /etc/nginx/ssl/rothchildbank.com/rothchild.priv
  5. For more accurate details and complete steps please refer to Nginx documentation or respective documentation for any other web-server.
