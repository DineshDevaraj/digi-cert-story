{{ page }}. MicroCert's Private & Public Key.

  1. Generate private key. openssl genrsa -out microcert.priv 2048
  2. Extract public key from private key. openssl rsa -in microcert.priv -pubout -out microcert.pub
  3. View the content of both private & public using cat, vim or any other text editor.
    cat microcert.priv
    cat microcert.pub
