{{ page }}. Part-2 Session Key Exchange

  1. Using rothchild.pub1 encrypt aes-256-key.bin into aes-256-key.enc and share with web-server 
    openssl pkeyutl -encrypt -pubin -inkey rothchild.pub1 \
    -in aes-256-key.bin -out aes-256-key.enc
  2. Web-browser receives aes-256-key.enc and decrypt using rothchild.priv to get AES key 
    openssl pkeyutl -decrypt -inkey rothchild.priv \
    -in aes-256-key.enc -out aes-256-key.bin1
  3. Note aes-256-key.bin will be equal to aes-256-key.bin1 and From now on every communication between web-client and web-server with be encrypted with aes-256-key.bin cmp aes-256-key.bin1 aes-256-key.bin
