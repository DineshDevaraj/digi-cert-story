{{ page }}. MicroAuth's Self-Signed Certificate.

  1. Now MicroAuth has a CSR. Next it should generate a self-signed certificate.
  2. The conept of self-signed certificate was discussed in the theoretical explanation for simplicity.
  3. Generate as self-signed certificate. openssl x509 -req -in micrcauth.csr -signkey micrcauth.priv -out micrcauth.crt -days 365
  4. View micrcauth.crt generated in previous step. openssl x509 -in micrcauth.crt -text -noout
  5. Next MicroAuth should publish the self-signed certificate to all the web-browsers.
  6. Now MicroAuth has all in place and it can start to serve customers.
