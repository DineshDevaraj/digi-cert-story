{{ page }}. Generate Digital Certificate

  1. After receiving the CSR (.csr file) MicroCert will start the process of signing the certificate. The result will be a rothchild.crt file that will be send back to the bank.
  2. Certain fields in the CSR are called (CRI) Certificate Request Information. Fields like applicant's-name, applicant's-address, applicant's-public-key and applicant's-signature are part of the CRI.
  3. Generate message-digest (hash) for CRI. Using MicroCert's private key generate MicroCert's-signature for this hash, add MicroCert's-signature to the rothchild.crt file and add MicroCert's public key to rothchild.crt file.
  4. Send rothchild.crt file to the bank. Note applicant's-signature and MicroCert's-signature are different, Applicant's public key and MicroCert's pulic key are different. Don't confuse between them.
